## 脚本功能
1.原有基础保留  
2.增加了划词添加到规则里面  
3.对abc-123这类关键字，做忽略大小写和'-'号  
4.可直接导入txt文件,会把换行和逗号做分割,做成一条规则  
5.划词功能设置开关
